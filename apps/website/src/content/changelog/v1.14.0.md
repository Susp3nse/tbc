### Build & Display System

- **Managed Multi-Group Display Scaffold** — The TMW profile template moved from a single icon group to a managed multi-group layout (Shown Main / Hidden APL / Visible APL). This is shared display plumbing every class builds on, so a rotation can expose a hidden action-priority group plus a toggle-gated visible display without each class hand-maintaining its own group structure.
- **Scaffold Refreshes on Every Sync** — The build now refreshes the managed group structure (groups, group count, texture) into existing profiles on every sync, not just on first creation, so older profiles upgrade to the new layout automatically. Note: this means in-game manual tweaks to group scale/columns/position are overwritten on the next sync — the group layout is build-managed.
