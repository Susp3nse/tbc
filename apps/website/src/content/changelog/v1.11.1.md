### Shaman — Enhancement

- **Twist Cycles Now Self-Correct** — The Windfury / Grace of Air and Fire Nova / Magma twist state machines tracked recommended drops rather than actual ones. If a recommendation window was missed (rotation icon shown but macro not pressed in time), the cycle desynced — symptoms included back-to-back Windfury drops with no Grace between and ~30s gaps with nothing dropping. Both twists now read `GetTotemInfo` each frame and act on what's actually deployed, so a missed window is recovered on the next frame automatically.
