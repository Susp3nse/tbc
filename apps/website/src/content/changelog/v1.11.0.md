### Shaman — Enhancement

- **Swing Sync Detection** — CLEU-based MH/OH tracker that reports stagger state on the combat dashboard (`OH lead` / `OK` / `Drift`). Helps you stay inside Flurry's 0.5s shared-charge window.
- **Auto Swing Resync (opt-in)** — Optional auto-fire of the resync macro (`/cleartarget` + `/targetlasttarget` + `/startattack`) when stagger drifts outside the Flurry window. Hysteresis dead-zone and swing-midpoint gating to avoid no-op fires. Off by default; most players will prefer to watch the dashboard and press manually.
- **WF / Grace Twist Rate Fix** — Twist phases were both holding for `CYCLE_TIME = 10s`, producing a 20s full loop instead of ~10s — roughly half the rate intended. Replaced with phase-specific durations (WF 1.5s, Grace 7.5s) so WF re-drops just before the carried 10s buff expires.
- **Post-FNT Fire Totem Dropdown** — New `enh_fnt_post_totem` setting (default Magma) controls the totem dropped between Fire Nova explosions. Lets you keep Nova → Magma → Nova cadence without forcing Magma as the single-target default.

### Shaman — Elemental / Enhancement

- **Hold Shocks (Interrupts Only)** — New `ele_shock_interrupt_only` / `enh_shock_interrupt_only` toggles suppress rotation shocks so the interrupt middleware can reserve them for kicks. Useful in dungeons.

### Shaman — All Specs

- **Grounding & Tremor Totems** — Now selectable as the default air / earth totem on every spec. Auto-Tremor middleware behavior unchanged.
